## For Students: How to Use Auto-Loading Links

1. **Click the link** your instructor provides
2. **Wait for data to load** (usually 1-2 seconds)
3. **See the data preview** - Verify it looks right
4. **Start analyzing!** Click on analysis tabs to begin

## Troubleshooting Checklist

| Problem | Solution |
|---------|----------|
| Link doesn't work | Verify Google Sheet is published as CSV |
| Data doesn't load | Check internet connection, disable ad blockers |
| Data looks wrong | Check original Google Sheet for errors |
| Can't find link | Ask instructor for the auto-loading link |
